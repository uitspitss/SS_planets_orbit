これは太陽系の惑星を図示するスクリプトです。

## 必要な物(pythonライブラリ)
* numpy
* matplotlib
* dateutil

## 使い方
`python planetary_orbits.py yyyy-mm-dd`

or

`python planetary_orbits_with_haya2.py yyyy-mm-dd`

を実行すると、同ディレクトリ内に画像ファイルが生成されます。
